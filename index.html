《LESS: Label-Efficient Semantic Segmentation for LiDAR Point Clouds》读书笔记

[TOC]

感觉这篇文章标题起的很好hhh，Less is more！论文起个好名字确实重要！

原文链接：https://arxiv.org/abs/2210.08064

在阅读这篇文献之前，如果你对点云分割的方法（采用全监督、弱监督、无监督还是半监督？）没有一个基础的概念的话，可以阅读一下这篇推送：https://mp.weixin.qq.com/s/a1a2-RjMC4IJW6WZVw5iJw

Abstract

Lidar的点云语义分割任务在自动驾驶中很重要，但是训练很深的网络需要很大的数据集，而标注数据集又很贵。

所以作者决定使用Label-Efficient的方法，使用半/弱监督学习（semi/weakly supervised learning）并且几乎适用于所有的3D语义分割的backbone。

具体来说，作者利用户外场景中的几何模式（geometry patterns）进行启发式预分割（heuristic pre-segmentation），减少人工标注，并与标注过程共同设计学习目标。在学习步骤中，我们利用原型学习（prototype learning）来获得更多描述性的点嵌入（point embeddings），并使用多扫描蒸馏（multi-scan distillation）从时间聚合点云（temporally aggregated point clouds）中开发更丰富的语义，以提高单扫描模型的性能。

RaLiBEV 读书笔记

发表于 2022-12-15 分类于 3D-Detection ， Lidar

《RaLiBEV: Radar and LiDAR BEV Fusion Learning for Anchor Box Free Object Detection System》读书笔记

[TOC]

Abstract & Introduction

雷达是唯一能够在所有天气条件下以可承受的成本提供可靠感知能力的传感器，已被广泛作为现代高级驾驶员辅助系统（ADAS）和自动驾驶系统中相机和激光雷达的关键补充。最近最先进的研究表明，雷达和激光雷达的融合可以在恶劣天气（如雾）下实现鲁棒检测。然而，这些方法仍然存在box估计精度低的问题。

本文提出了一种用于anchor box-free目标检测系统的BEV融合学习方法，该方法利用雷达距离-方位热图（range-azimuth heatmap）和激光雷达点云的特征来估计可能的目标。

设计了不同的标签分配策略，以促进前景或背景anchor point的分类与相应的边界框回归之间的一致性。此外，通过采用新颖的交互式transformer模块，可以进一步提高所提出的目标检测器的性能。论文使用最近发布的Oxford Radar RobotCar（ORR）数据集展示了所提出方法的优越性能。实验表明，RaLiBEV的精度大大优于其他最先进的方法。

Pointnet(++)学习笔记

[TOC]

PointNet

Abstract

本文的贡献在于不像之前的工作将点云转换为规则的三维提速网格或图像集合，而是直接以点的形式处理点云，并且很好的尊重了点云的排列不变性。

在这方面的探索，Pointnet的工作是开创性的，而且它简洁、高效、强大。

PointNet所作的事情就是：对点云做特征学习，并将学习到的特征去做不同的应用：分类（shape-wise feature）、分割（point-wise feature）等。

PointNet之所以影响力巨大，就是因为它为点云处理提供了一个简单、高效、强大的特征提取器（encoder），几乎可以应用到点云处理的各个应用中，其地位类似于图像领域的AlexNet。

（当然，Alexnet带来的影响过了几年人们又觉得有一部分可能是不太好的，但是探索开创的意义重大）

DETR 论文精读

发表于 2022-11-15 更新于 2022-12-15 分类于 3D-Detection

DETR论文精读(eccv20 Transfomer+目标检测)

[^9000+stars，very good]:

简介：end-to-end的里程碑意义很大，因为以前的目标检测领域很少有end-to-end的方法，大部分方法都需要一个后处理的操作，即nms（non-maxinum suppersion，非极大值抑制），无论是 proposal based的方法，还是anchor based的方法，还是anchor free的方法，最后都会生成很多预测的框框。nms的作用就是去掉这些框框，这些框框也会对模型的调参和部署产生一些影响。而且nms这个操作也不是所有硬件都支持的。

而DETR解决了上面提到的这些痛点，既不需要proposal，也不需要anchor，直接利用transfomer全局建模的能力，把目标检测看成了集合预测的问题，也不会输出那些冗余的框框。把目标检测任务做到了和图像分类一样简单。（nms和anchor的生成这两部分都需要不少的先验知识）

《A comprehensive survey of LIDAR-based 3D object detection methods with deep learning for autonomous driving》读书笔记

发表于 2022-11-15 更新于 2022-12-15 分类于 3D-Detection

《A comprehensive survey of LIDAR-based 3D object detection methods with deep learning for autonomous driving》 （《基于LIDAR的自动驾驶深度学习3D物体检测方法的综合调查》 ）读书笔记

[TOC]

Section1 引言

与摄像头相比，LiDAR不受光照条件的限制，而且以无隐私的方式捕获数据，这使得它们在技术和道德方面都非常适合复杂的户外环境。这篇文章关注的是：基于LiDAR的3D目标检测器（仅使用LiDAR传感器数据，而不依赖任何其他传感器的数据）

针对点云中的3D目标检测及其在自动驾驶中的应用已有大量研究，其他综述类文献侧重于自动驾驶应用中的目标检测和语义分割，以及使用多模态技术的深度学习网络。

与前述工作相比，该综述的贡献总结如下：

1.文章介绍了一个通用操作流程，为结构化分类奠定基础，便于比较并显示各检测器间的相似之处和不同之处。

2.文章提供了详尽的最新3D目标检测器列表。侧重于描述每种目标检测器的优缺点和其在操作流程上的特殊性，以确保检测结果的高效性和有效性。

3.文章最终的结论确定了未来新的3D目标检测器设计中应该采用或避免的一些关键特征。
